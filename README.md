# 🔐 HackAcademy — Ethical Hacking Platform

A gamified ethical hacking learning platform with a live Linux terminal simulator, XP system, quizzes, and module-based curriculum.

---

## 🚀 Quick Deploy to Vercel (Recommended)

1. Fork or upload this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Sign in with GitHub
3. Click **Add New Project** → Import your repo
4. Click **Deploy** — done in 2 minutes!
5. On mobile: open the URL → Share → **Add to Home Screen**

---

## 💻 Run Locally

```bash
npm install
npm start
```

Then open http://localhost:3000

---

## 📱 Install as Mobile App (PWA)

After deploying to Vercel:
- **Android**: Open in Chrome → 3-dot menu → **Add to Home Screen**
- **iPhone**: Open in Safari → Share button → **Add to Home Screen**
